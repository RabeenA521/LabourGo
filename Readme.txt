# LabourGo

## Overview

LabourGo is a full-stack labour hiring platform that connects customers with skilled workers through web and mobile applications.

## Technologies Used

- Flutter
- Django REST Framework
- ASP.NET MVC
- SQL Server
- JWT Authentication
- Postman
- Git & GitHub

## Features

- User Registration and Login
- JWT Authentication
- Worker Discovery
- Booking Management
- Customer Dashboard
- Worker Profiles
- REST API Integration
- Real-Time Messaging
